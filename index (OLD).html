My Process

efficency is key
01

Define Project

The Project Definition outlines the general overview of the project. This includes defining our goals and core messages while catering to our target audience.

02

Wireframes and Site Architecture

Creating a site architecture, sitemap and wireframe will provide a detailed view of the content that will appear on each page, and the relationship each individual page has with one another.

03

Visual Design

Once the website blueprint has been defined, we'll have to determine what the visual brand of the organization will look like. This includes colors, fonts, design character, and more.

04

Site Development

Now that we've prepped the website, it's now time to develop the actual pages. Site Development includes writing code (HTML, CSS, JS, etc.), refining content, and media creation.

05

Site Testing

After the website is developed, it will generally be placed on a private server for testing. Content reviewers will be granted access to the server to locate and report any issues created in the development process.

06

Launch

Once the website has been approved by the content reviewers, it's time to officially launch. If requested, I can and will continue to maintain the website. This service will be discussed further at the time of launch.
